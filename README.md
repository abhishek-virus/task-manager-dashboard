# Advanced Task Manager Dashboard 🚀

A modern and interactive Task Manager Dashboard built using HTML, CSS, and JavaScript. This project was created to make task 
organization simple and visually engaging while also improving my frontend development skills.

The application allows users to create, manage, organize, and track tasks with useful features like drag & drop, dark mode,
progress tracking, search functionality, and local storage support.

---

## ✨ Features

- Add new tasks with title, description, priority, and due date
- Edit existing tasks anytime
- Delete tasks
- Mark tasks as completed
- Search tasks instantly
- Sort tasks based on priority
- Drag and drop tasks to reorganize them
- Progress bar to track completion percentage
- Dark/Light mode toggle
- Automatic local storage support
- Responsive design for different devices

---

## 🛠 Tech Stack

Frontend:

- HTML5
- CSS3
- JavaScript (ES6)

## How It Works

### Adding a Task
Enter:

- Task title
- Description
- Priority
- Due date

Then click:

```text
Add Task
```

---

### Editing Tasks

Click the **Edit** button to update task information.

---

### Completing Tasks

Use the checkbox to mark tasks as completed.

The progress bar automatically updates according to completed tasks.

---

### Reordering Tasks

Drag and drop task cards to rearrange them according to your preference.
