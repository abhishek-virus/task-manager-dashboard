# Advanced Task Manager Dashboard 🚀

Live:- 

A modern and interactive Task Manager Dashboard built with **HTML, CSS, and JavaScript**. This project helps users manage daily tasks in an organized way with features like task creation, editing, search, drag & drop, dark mode, and progress tracking.

I built this project to improve my frontend development skills and gain hands-on experience with JavaScript concepts like DOM manipulation, local storage, event handling, and dynamic UI updates.

---

## Features

✅ Add new tasks

✅ Edit task details

✅ Delete tasks

✅ Mark tasks as completed

✅ Set task priorities (Low, Medium, High)

✅ Add due dates

✅ Search tasks instantly

✅ Sort tasks by priority

✅ Drag and drop task cards

✅ Dark/Light mode toggle

✅ Dynamic progress bar

✅ Local storage support

✅ Fully responsive design

---

## How It Works

### Adding a Task

Users can:

- Enter task title
- Write a description
- Select priority level
- Add due date
- Click **Add Task**

The task instantly appears on the dashboard.

---

### Editing Tasks

Tasks can be updated anytime using the **Edit** button.

An edit modal opens where users can modify:

- Title
- Description
- Priority
- Due date

---

### Search Feature

The search bar filters tasks in real time, making it easier to find specific tasks quickly.

---

### Task Progress

The dashboard automatically tracks completed tasks and updates the progress bar based on completion percentage.

---

### Dark Mode

Users can switch between light mode and dark mode for a better viewing experience.

---

### Data Storage

This project uses browser **Local Storage**, which means tasks remain saved even after refreshing or closing the browser.

No backend or database is required.
